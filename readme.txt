== Installation ==

1. Upload the entire `woocommerce-tl-birimi` folder to the `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress

3. Look under the products

4. Go and manage your orders - good luck with sales :)



**Please note:** You must run WordPress 3.3 or higher and WooCommerce 1.4 or higher in order tun this plugin. This is due to changes in WooCommerce v1.4+!
